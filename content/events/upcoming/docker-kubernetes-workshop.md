## Workshop Overview

Learn the fundamentals of **Docker and Kubernetes** in this hands-on 3-hour workshop! Whether you're new to containerization or looking to strengthen your DevOps skills, this workshop will give you practical experience with industry-standard tools.

### What You'll Learn

- **Docker Basics**: Containers, images, Dockerfiles, and Docker Compose
- **Containerizing Applications**: Turn any application into a Docker container
- **Multi-Container Apps**: Connect multiple containers using Docker Compose
- **Kubernetes Introduction**: Pods, services, deployments, and basic orchestration
- **Real-World Use Cases**: When and why to use containers in production

### Workshop Format

This is a **hands-on, interactive workshop**. You'll:
- Follow along with live coding demonstrations
- Complete practical exercises on your own laptop
- Build and deploy containerized applications
- Ask questions and get help from the instructor
- Take home complete code examples and reference materials

## Installation Instructions

Please install Docker Desktop **before the workshop** to save time:

### Windows/Mac
1. Download Docker Desktop from [docker.com](https://www.docker.com/products/docker-desktop/)
2. Run the installer
3. Verify installation: Open terminal and run `docker --version`

### Linux
```bash
# Ubuntu/Debian
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER
```

**Having installation issues?** Join our Discord #workshop-help channel or come 15 minutes early for setup assistance.
