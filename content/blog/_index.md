## Contributing to the Blog

We encourage all club members to share their knowledge! Whether you're a beginner documenting your learning journey or an expert sharing advanced concepts, your voice matters.

{% collapse(title="How to Write a Blog Post:") %}
1. Check out our example blog posts in the `blog/2025/` folder
2. Fork the repository and create your markdown file
3. Follow the [contribution guidelines](https://github.com/your-org/cc-website/blob/main/CONTRIBUTING.md)
4. Submit a pull request
{% end %}
